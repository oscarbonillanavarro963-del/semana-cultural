Semana Cultural 2026

Página web informativa para la Semana Cultural 2026, un evento con conciertos, teatro y talleres gratuitos.

Secciones usadas y columnas de Bootstrap
- Navbar: menú de navegación
- Hero: sección principal con imagen de fondo
- Sección 1 - Información del evento
- Sección 2 - Actividades que se realizan en el evento
- Footer: 3 - Información de contacto

Componentes de Bootstrap usados
- Navbar
- Cards
- Carrusel
- Botones
- Grid system

Dificultad encontrada y solución
La mayor dificultad fue entender cómo funcionan las columnas de Bootstrap. Se solucionó consultando la documentación y probando diferentes combinaciones hasta conseguir el diseño que mas me gustase.
