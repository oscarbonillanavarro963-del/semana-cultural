Semana Cultural 2026

Descripción del proyecto
Página web informativa para la Semana Cultural 2026, un evento con conciertos, teatro y talleres gratuitos.

Secciones usadas y columnas de Bootstrap
- Navbar: menú de navegación
- Hero: sección principal con imagen de fondo
- Sección 1 - Información: 3 columnas (col-md-4) con tarjetas
- Sección 2 - Actividades: 3 columnas (col-md-4) con imágenes
- Footer: 3 columnas con información de contacto

Componentes de Bootstrap usados
- Navbar
- Cards
- Carrusel
- Botones
- Grid system

La mayor dificultad fue entender cómo funcionan las columnas de Bootstrap. Se solucionó consultando la documentación y probando diferentes combinaciones hasta conseguir el diseño deseado.
